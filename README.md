 这个仓库收集了来自3个不同仓库的中文谣言数据集，它们分别是。
 
 1.Chinese_Rumor_Dataset(https://github.com/thunlp/Chinese_Rumor_Dataset)

 该数据为从新浪微博不实信息举报平台抓取的中文谣言数据。

 2.DoubleCheck(https://github.com/Enderfga/DoubleCheck)

 该数据集LTCR为准确检测错误信息提供了宝贵的资源，特别是在与COVID-19相关的复杂假新闻的背景下。由于CSV文件不适用于模型训练，可参考data\raw\DoubleCheck下的conversion.py进行格式转换。

 3.CHECKED(https://github.com/cyang03/CHECKED)
 
 为与COVID-19相关的中文谣言数据集，以json格式与csv格式存储。
 再次声明：这些数据源并非来自本人收集，只是针对上诉仓库的整理。
